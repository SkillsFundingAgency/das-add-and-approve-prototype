Upload multiple apprentices

Use our csv template to upload details for multiple apprentices at the same time. The file must contain mandatory information in a specific format. This information and formatting includes:

  • a unique learner number (ULN) that’s 10-digits long. The number 9999999999 is invalid
  • Date of birth formatted as YYYY-MM-DD. You will need to set this date format manually when using Excel
  • Start and end dates formatted as YYYY-MM
